## Description
The web application allows you, to save tasks to be done soon on the board.  
The green button allows you to mark the task as completed. 
Each of the tasks can be removed from the board at any time.

## Technologies
- HTML
- CSS
- BEM convention
- Javascript
- ES6+ features
